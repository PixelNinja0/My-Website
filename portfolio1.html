AI-Powered PDF Reader

An accessible, AI-powered PDF reader designed to help blind and visually impaired users interact with PDF content using natural language. Users can ask questions about a PDF and receive instant answers, optionally read aloud via text-to-speech. Developed as a university group project, the tool combines a clean web interface with smart AI integration.

My Role

  • Team Lead & Fullstack Developer
  • Implemented both backend (FastAPI) and frontend (HTML, CSS, JavaScript)
  • Integrated AI features (OpenAI GPT) and local text-to-speech

Highlights

  • Simple, browser-based, accessible interface
  • Natural language queries for PDF content
  • Dummy mode for testing without API key
  • Local text-to-speech using pyttsx3

Tech Stack
